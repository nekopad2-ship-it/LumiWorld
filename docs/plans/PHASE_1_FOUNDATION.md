# Phase 1 Foundation

This repository is currently implementing Phase 1 only:

- Spindle scaffold and manifest
- shared schemas and contracts
- `WorldGraph` schema v3
- transactional patch service with idempotency
- per-chat serialization and generation correlation
- storage and migration foundations
- floating orb
- empty Tracker Dock
- settings drawer
- static `LWE_SCENE_IMPACT` injection
- build, typecheck, lint, unit, and integration test setup

Explicitly deferred:

- Profile Builder
- State Extractor
- Time Inference
- NPC Agency Processor
- autonomous NPC behavior
- profile inference
